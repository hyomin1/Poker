## 시작 가이드
### Requirements
- [Node.js 18.17.1](https://nodejs.org/en)
- [Npm 8.19.3](https://www.npmjs.com/package/npm/v/8.19.2)

### Installation
``` bash
$ git clone https://github.com/hyomin1/poker.git
$ cd poker
```
#### Frontend
```
$ cd poker
$ npm install 
$ npm start
```
